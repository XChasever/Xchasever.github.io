博客的源代码

要求
node.js - v22.22.2

下载
vscode or others
git clone  xxx
use branch code
npm install hexo-cli -g
npm install hexo-filter-mathjax

使用
npm install 
修改source/_posts/xxx.md、/_config.yml、/themes/redefine/_config.yml
hexo server

部署
hexo clean
hexo deploy
